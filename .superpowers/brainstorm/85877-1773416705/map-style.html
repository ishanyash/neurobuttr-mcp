How should the relationship map feel?

This is the core visual — the thing you stare at while working. Pick the vibe.

A: Node Graph

Force-directed graph like a mind map. Nodes float freely, connected by edges. Organic, spatial. Think: Obsidian graph view.

B: Git Graph

Vertical timeline with branches forking off and merging back. Linear, familiar to developers. Think: GitLens, git log --graph.

C: Metro Map

Horizontal flowing lines like a subway map. Branches curve off and rejoin. Clean, modern, distinctive. Think: London Tube map meets code.
